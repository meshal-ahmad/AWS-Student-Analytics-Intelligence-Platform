
# AWS Student Analytics Intelligence Platform

📌 Project Overview
This project demonstrates how AWS cloud analytics services can be used to transform raw student enrollment data into meaningful business insights.  
The goal is to support academic and financial decision-making using data-driven dashboards.

---

 Technologies Used
- Amazon S3 – Data Storage
- Amazon Athena – SQL Querying
- Amazon QuickSight – Data Visualization
- CSV Dataset – Student Enrollment Data

---

 Cloud Architecture Workflow
1. Upload raw dataset to Amazon S3  
2. Query data using Amazon Athena (SQL)  
3. Visualize insights using Amazon QuickSight dashboards  

---
 Key Dashboards & Insights


 Enrollment Trends
- Identify most popular majors
- Track academic demand changes over time
- 

 Course Evaluation Analysis
- Detect highest rated courses
- Identify successful teaching strategies
- 

 Course Cost Analysis
- Identify most expensive courses
- Support budget optimization decisions


 Student Demographics Analysis
- Youth vs Adult student distribution
- Support flexible program planning
